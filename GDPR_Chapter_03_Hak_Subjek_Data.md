# Bab 3: Hak Subjek Data (Rights of the Data Subject)

GDPR memberikan kontrol penuh kepada individu atas data pribadi mereka.

### 1. Hak untuk Diinformasikan (Art. 12-14)
- Individu harus diberi tahu tentang bagaimana data mereka dikumpulkan dan digunakan (melalui Privacy Policy).

### 2. Hak Akses (Art. 15)
- Individu berhak mendapatkan konfirmasi apakah data mereka sedang diproses dan mendapatkan salinan data tersebut.

### 3. Hak Perbaikan / Rectification (Art. 16)
- Individu berhak meminta koreksi atas data yang tidak akurat atau melengkapi data yang tidak lengkap.

### 4. Hak Penghapusan / Right to be Forgotten (Art. 17)
- Individu dapat meminta penghapusan data mereka jika:
  - Data tidak lagi diperlukan.
  - Subjek menarik persetujuan (consent).
  - Data diproses secara melawan hukum.

### 5. Hak Pembatasan Pemrosesan (Art. 18)
- Subjek dapat meminta penghentian sementara pemrosesan data mereka (misalnya saat memverifikasi akurasi).

### 6. Hak Portabilitas Data (Art. 20)
- Individu berhak menerima data mereka dalam format yang terstruktur, umum digunakan, dan dapat dibaca mesin (seperti JSON/CSV) untuk dipindahkan ke layanan lain.

### 7. Hak Keberatan / Object (Art. 21)
- Individu berhak menolak pemrosesan data untuk kepentingan pemasaran langsung (direct marketing) atau berdasarkan kepentingan legitimasi (legitimate interest).

### 8. Hak Terkait Pengambilan Keputusan Otomatis (Art. 22)
- Individu berhak untuk tidak tunduk pada keputusan yang sepenuhnya berdasarkan pemrosesan otomatis (seperti profiling) yang memiliki dampak hukum signifikan.
